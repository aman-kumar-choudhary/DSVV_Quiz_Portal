Quiz Instructions

Please read all instructions carefully before starting the quiz

Time Limit

Each quiz has a specific time limit. The timer will be displayed throughout the quiz.

  • Timer turns yellow when 5 minutes remain
  • Timer turns red and pulses when 2 minutes remain
  • Quiz auto-submits when time expires

Question Navigation

Use the question panel to navigate between questions.

  • Answered correctly
  • Answered incorrectly
  • Skipped question

Skipping Questions

You can skip questions and return to them later.

  • Skipped questions can be reattempted
  • Answered questions cannot be changed
  • All questions must be attempted or skipped before finishing

Important Notes

Please note the following important points:

  • Don't refresh the page during the quiz
  • Don't navigate away from the quiz page
  • Internet connection must remain stable
  • Quiz will auto-submit when time expires
